# Запуск и сборка
``` 
 make 
 ./main -flags
```

| Флаг    | Полный ключ       | Описание                                                                                                                                     |
|---------| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| `-exit` | `flags.Exit`      | Завершает горутину вызовом `runtime.Goexit()` (принудительный выход).                                                                        |
| `-ctx`  | `flags.Ctx`       | Использует `context.Context` для управления завершением горутины.                                                                            |
| `-after` | `flags.After`     | Останавливает горутину с использованием `time.After` (по таймауту).                                                                          |
| `-cond` | `flags.Condition` | Завершает горутину через логический флаг или условие (`bool`, `atomic`, и т.п.).                                                             |
| `-t`    | `flags.Time`      | Включает режим с таймаутом при использовании `-ctx`. Используется **в паре** с `-ctx`. Вызывает `CtxWithTimeout(3)` вместо `CtxWithCancel()` |
